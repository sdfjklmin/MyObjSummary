    1.技术诞生的背景是什么，这是理解技术理念的关键。
    2.技术的优势和劣势分别是什么，技术没有银弹，只有适应场景，在某一方面有优势，那在其他方面就肯定有劣势。

    4.和已有的其他技术相比有什么不同
